### thrift_demo 
实践了官方的node.js和python的客户端和服务器，并且可以相互调用，python客户端调用node.js服务器。


### node.js server (thrift包请安装好)
cd node_thrift

node tf_server.js

### node.js client
cd node_thrift

node tf_client.js


### python server (thrift包请安装好)
cd tf_python

python tf_server.py


### python client 
cd tf_python

python tf_client.py
